# Introduction to Programming Using C++ – Level 2

This repository contains all code examples, exercises, and projects for the **C++ Level 2** course. This stage focuses on advanced concepts, memory control, and professional programming skills.

## 🗝️ About This Course
Advanced C++ Concepts, Control, and Professional Programming Skills. This course bridges the gap between writing code that "just works" and understanding exactly how and why it works in memory.

## 📖 Learning Outcomes
* **Professional Debugging:** Using breakpoints, watches, and inspecting execution flow.
* **Memory Management:** Mastering pointers, references, stack vs. heap, and dynamic allocation (`new`/`delete`).
* **Advanced Data Structures:** In-depth use of Vectors and Iterators.
* **File Handling:** Writing, reading, updating, and deleting records in files.
* **Core Concepts:** Function overloading, recursion, static variables, and exception handling.
